# Port Scanner

A simple Python port scanner that checks a target IP address or domain for open ports within a specified range. Supports both normal and verbose modes, and maps ports to service names.

## Features
- Scan any IP address or domain  
- Custom port range scanning  
- Returns list of open ports or a formatted verbose report  
- Handles invalid hostname or IP address  

## Files
- `port_scanner.py` — main scanning function  
- `common_ports.py` — dictionary of common ports and service names  
- `main.py` — example/test script
